# Data Warehouse Project

This project demonstrates the design and implementation of a data warehouse using PostgreSQL, following best practices in data modeling and transformation.

---

## 📦 Project Setup

### 1. Install PostgreSQL
Make sure you have PostgreSQL installed and running on your machine.

---

### 2. Connect to PostgreSQL Server

Using your database client (e.g., DataGrip or psql), create a connection to the PostgreSQL server:

- Host: localhost  
- Port: 5432 (default)  
- User: your PostgreSQL username  
- Password: your PostgreSQL password  
- Database: postgres (default database)

---

### 3. Create the Database

After connecting, run:

```sql
CREATE DATABASE dwh_project;
